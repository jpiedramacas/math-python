Enunciado:
Desarrollar un programa que demuestre conceptos básicos de seguridad informática en Python, como cifrado y descifrado de datos, hashing de contraseñas y prevención de ataques.

Explicación:
La seguridad informática es un aspecto fundamental en el desarrollo de software para proteger la confidencialidad, integridad y disponibilidad de los datos. Algunos conceptos básicos de seguridad informática incluyen el cifrado y descifrado de datos para proteger su confidencialidad, el hashing de contraseñas para almacenarlas de forma segura y la prevención de ataques como la inyección de código y la denegación de servicio.

En Python, podemos utilizar bibliotecas como `cryptography` para el cifrado y descifrado de datos, y `hashlib` para el hashing de contraseñas. Estas bibliotecas proporcionan funciones y herramientas para implementar prácticas de seguridad informática de manera efectiva.

El enunciado solicita desarrollar un programa que demuestre estos conceptos básicos de seguridad informática en Python. Esto podría incluir la generación de claves de cifrado, el cifrado y descifrado de mensajes, el hashing de contraseñas y la verificación de contraseñas hash.

Es importante comprender estos conceptos y cómo implementarlos correctamente en tus aplicaciones para garantizar la seguridad de los datos y proteger contra posibles ataques y vulnerabilidades.
