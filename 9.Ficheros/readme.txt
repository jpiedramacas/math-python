El enunciado es el siguiente:

Ficheros:

Escribir un programa que lea y escriba en ficheros en Python.

Incluir ejemplos de lectura y escritura en ficheros, así como recorrido de un fichero secuencial utilizando feof.

Explicación:

Este ejercicio se centra en manipular archivos en Python. El programa debe ser capaz de realizar las siguientes operaciones:

1. Escribir en un archivo: Se deben incluir ejemplos de cómo escribir datos en un archivo de texto.
2. Leer desde un archivo: Se debe demostrar cómo leer datos desde un archivo de texto.
3. Recorrer un archivo secuencialmente: Se debe mostrar cómo recorrer un archivo línea por línea utilizando un bucle while y la función `feof()` para detectar el final del archivo.

Estas operaciones son fundamentales para interactuar con archivos en Python y son útiles en una variedad de escenarios de programación.
