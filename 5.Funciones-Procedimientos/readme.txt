El enunciado del ejercicio de Funciones y Procedimientos es el siguiente:

**Funciones y Procedimientos:**
Crear funciones y procedimientos en Python que realicen tareas específicas, con y sin parámetros. Incluir ejemplos que demuestren parámetros por valor y por referencia.

**Explicación:**
En este ejercicio, se solicita crear funciones y procedimientos en Python para realizar diversas tareas. Las funciones son bloques de código que realizan una tarea específica y pueden devolver un valor, mientras que los procedimientos son bloques de código que realizan una tarea pero no devuelven ningún valor. 

En el código proporcionado, se definen varias funciones y procedimientos para realizar diferentes tareas, como sumar dos números, imprimir un mensaje, modificar una lista, duplicar una lista y intercambiar los valores de dos variables. Además, se incluyen ejemplos que muestran cómo se pasan los parámetros por valor y por referencia en Python.
