Enunciado:

Desarrollar un programa que utilice los distintos operadores aritméticos y lógicos en Python.

Explicación:

El programa muestra ejemplos del uso de operadores aritméticos y lógicos en Python. 

- **Operadores Aritméticos:** Se utilizan para realizar operaciones matemáticas básicas, como la suma (`+`), resta (`-`), multiplicación (`*`), división (`/`), módulo (`%`), y potencia (`**`). Se ejemplifica el uso de estos operadores con números enteros.

- **Operadores Lógicos:** Se utilizan para realizar operaciones lógicas entre expresiones booleanas. Los operadores lógicos más comunes son `and`, `or` y `not`. Se demuestra el uso de estos operadores con variables booleanas.

El programa imprime los resultados de las operaciones aritméticas y lógicas realizadas, proporcionando ejemplos claros del uso de estos operadores en Python.
