Enunciado:
Desarrolla un programa en Python que emplee la programación orientada a objetos (POO) para modelar entidades del mundo real. Utiliza clases, objetos y métodos para representar las características y comportamientos de estas entidades.

Explicación:
La programación orientada a objetos (POO) es un paradigma de programación que permite modelar entidades del mundo real como objetos con características (atributos) y comportamientos (métodos). En Python, las clases son la base de la POO, y los objetos son instancias de estas clases.

Para utilizar la POO en Python, primero debes definir una clase. Una clase es como un plano o una plantilla que define las características y comportamientos de un tipo específico de objeto. Los atributos de una clase son variables que almacenan datos asociados a cada objeto, mientras que los métodos son funciones que definen las acciones que pueden realizar los objetos de esa clase.

Una vez que has definido una clase, puedes crear objetos (instancias) de esa clase utilizando el constructor de la clase. Luego, puedes acceder a los atributos y métodos de los objetos utilizando la notación de punto.

En el enunciado se te pide desarrollar un programa que utilice la POO para modelar entidades del mundo real. Puedes elegir cualquier entidad que desees modelar, como personas, animales, vehículos, etc. Debes definir una o más clases para representar estas entidades, especificando sus atributos y métodos según sea necesario.

Al emplear la POO, tu programa será más modular, legible y reutilizable, ya que podrás encapsular la funcionalidad relacionada en clases y objetos independientes. Además, la POO te permite abstraer los detalles de implementación y centrarte en el comportamiento y las interacciones entre las entidades modeladas.
