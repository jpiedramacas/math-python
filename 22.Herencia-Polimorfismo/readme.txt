Enunciado:
Desarrolla un programa en Python que demuestre el uso de herencia y polimorfismo. Debes crear una jerarquía de clases donde una clase base tenga al menos un método que será sobrescrito en sus subclases. Utiliza las subclases para mostrar cómo el polimorfismo permite llamar al método adecuado según el tipo de objeto.

Explicación:
La herencia es un concepto clave en la programación orientada a objetos que permite la creación de clases nuevas basadas en clases existentes. Una clase que hereda de otra se llama subclase, y la clase de la que hereda se llama clase base o superclase. La herencia permite la reutilización del código y la creación de una jerarquía de clases que refleja relaciones entre diferentes tipos de objetos.

El polimorfismo es otro concepto importante que se deriva de la herencia. Se refiere a la capacidad de diferentes tipos de objetos de responder al mismo mensaje (llamada a un método) de manera diferente. Esto significa que un objeto puede tomar muchas formas. En Python, el polimorfismo se logra mediante la sobrescritura de métodos. Una subclase puede proporcionar su propia implementación de un método que ya está definido en una superclase.

En el enunciado se te pide desarrollar un programa que demuestre el uso de herencia y polimorfismo en Python. Debes crear una jerarquía de clases donde una clase base tenga al menos un método que será sobrescrito en sus subclases. Luego, utiliza los objetos de estas subclases para mostrar cómo el polimorfismo permite llamar al método adecuado según el tipo de objeto.

El ejemplo proporcionado en el enunciado ilustra cómo se pueden utilizar la herencia y el polimorfismo para crear una jerarquía de clases y cómo los objetos de estas clases pueden comportarse de manera diferente en función de su tipo. Esto demuestra cómo la programación orientada a objetos permite la creación de modelos más flexibles y extensibles en Python.
