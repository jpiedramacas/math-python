Enunciado:

Desarrolla un algoritmo en Python que siga un diagrama de flujo dado. Supongamos que el diagrama de flujo es para encontrar el máximo de tres números.

Explicación:

El objetivo de este ejercicio es implementar un algoritmo en Python que encuentre el máximo de tres números dados. La lógica para encontrar el máximo consiste en comparar los números entre sí y seleccionar el más grande.

El algoritmo se puede implementar utilizando una función que tome tres números como entrada y devuelva el máximo de los tres. En el código, se utiliza una serie de condicionales `if-elif-else` para comparar los números y determinar cuál es el máximo. Finalmente, se imprime el resultado obtenido.

Este tipo de algoritmo es común en la programación y es útil para una variedad de aplicaciones donde se necesite encontrar el valor máximo entre un conjunto de datos.
