**Enunciado:**  
Implementar un programa que maneje fechas en Python utilizando la biblioteca datetime.

**Explicación:**
Este programa muestra cómo trabajar con fechas y horas en Python utilizando la biblioteca estándar datetime. Utiliza la función `datetime.now()` para obtener la fecha y hora actual, y luego muestra cómo acceder a diferentes componentes de la fecha y la hora, como el año, el mes, el día, la hora, el minuto y el segundo. También muestra cómo crear una fecha específica utilizando la clase `datetime`, y cómo calcular la diferencia entre dos fechas para obtener la edad en años.
