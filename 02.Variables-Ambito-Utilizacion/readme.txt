El enunciado:
pide crear un programa que demuestre el uso de variables y su ámbito en Python. El ámbito se refiere al contexto en el que una variable es válida y puede ser accedida. Hay dos tipos principales de ámbito en Python: global y local.

El programa debe incluir lo siguiente:

1. Definición de una variable global.
2. Definición de una función que utilice una variable local.
3. Llamada a la función para mostrar cómo se accede a las variables dentro y fuera de su ámbito.

El ejemplo proporcionado define una variable global llamada `global_var` y una función llamada `test_function()`. Dentro de la función, se define una variable local llamada `local_var`. Luego, se llama a la función para mostrar el acceso a estas variables tanto dentro como fuera de la función.

Es importante comprender cómo funcionan los ámbitos en Python para evitar errores y escribir un código más legible y mantenible.
